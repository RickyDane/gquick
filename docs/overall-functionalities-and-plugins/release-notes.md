# Release Notes: Overall Functionalities and Plugins Documentation

## Summary
Added a current documentation hub for GQuick users and developers that explains the launcher features, built-in plugins, setup requirements, plugin development workflow, and known limitations.

## What's New
- **Feature Overview**: Documents core launcher, AI, capture/OCR, notes, Docker, weather, network, and speedtest functionality.
- **Plugin Catalog**: Lists current plugins with triggers, actions, and AI tool support.
- **Setup Notes**: Covers AI providers, OCR, Docker, shortcuts, weather, speedtest, and local development.
- **Developer Notes**: Explains how to add/update plugins and plugin tools.
- **Security Notes**: Highlights API key storage, external service calls, file-read safeguards, and platform limitations.

## Bug Fixes
- Clarifies stale documentation gaps where older docs omit newer plugins or current Settings behavior.

## Migration Notes
No migration required. This is a documentation-only update.

## Breaking Changes
None.
